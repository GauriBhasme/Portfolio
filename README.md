# Personal Portfolio Website

A modern, responsive one-page portfolio website built with HTML, CSS, and JavaScript.  
Designed to showcase skills, projects, and experience with dynamic rendering from structured data files.

---

## Features

- Clean modern UI design
- Fully responsive layout
- Dynamic rendering using JavaScript data files
- Skills section with progress indicators
- Project showcase cards
- Experience timeline/cards
- Contact section with resume download
- Modular and scalable structure

---

## Tech Stack

- HTML5
- CSS3 (custom styling + responsive layout)
- Vanilla JavaScript (dynamic rendering)



## How to Run Locally

1. Clone the repository

2. Open the project folder

3. Launch `index.html` in your browser

No build tools or dependencies required.

---

## How to Customize


Edit: `data.js` to add new data

## Responsiveness

The layout automatically adapts to:

- Desktop
- Tablet
- Mobile

No additional configuration needed.
